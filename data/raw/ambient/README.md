# data/raw/ambient

Raw audio clips representing **non-violent ambient/background scenes**. Used as the negative/background class.

## Current subfolders
- `bathroom/` — bathroom ambience (running water, fans, etc.).
- `brushing_teeth/` — toothbrushing sounds.
- `computer/` — keyboard typing, mouse clicks, fan/case noise.
- `door_footsteps/` — doors opening/closing, footsteps.
- `human_noises/` — non-speech human sounds (coughing, sneezing, breathing, etc.).
- `kitchen/` — kitchen ambience (appliances, dishes, room background).
- `noise&scraping/` — generic noise and scraping/rustling sounds.
- `pets/` — pet sounds (cats, dogs, etc.) in a non-aggressive context.
- `silence/` — quiet/near-silent room tone.
- `tv/` — TV/background audio that is not violence.
- `vacuum_cleaner/` — vacuum cleaner running.
